# RISC-V Architecture Test Program Environment
These files are a local implementation of the RISC-V Architecture Test Program Environment.
The files were copied from the `act4` branch of https://github.com/riscv-non-isa/riscv-arch-test/tests/env, hash eb18e5e.

## Using:
In theory, the only thing to do is add this directory to your include path and implement the TrickBox macros in model_test.h.
